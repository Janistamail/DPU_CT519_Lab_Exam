## ขั้นตอนการ run App_A

### Backend

<p>1. git clone https://github.com/Janistamail/DPU_CT519_Lab_Exam.git</p>
<p>2. cd เข้า App_A/backend </p>
<p>3. npm start</p>
<p>4. npm start</p>

### Database

<p>import database file (/App_A/mysql-backup/0030_Lab_Exam.sql) เข้า phpMyAdmin</p>
